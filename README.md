# 🌦️ Tirupati Live Weather Dashboard

## 📌 Project Overview

This project is a real-time weather dashboard built to monitor and analyze weather conditions in Tirupati. It provides live updates on temperature, humidity, wind speed, and other key metrics through an interactive and user-friendly interface.

The goal of this project is to demonstrate how real-time data can be integrated, visualized, and interpreted for practical use.

---

## 🌐 Live Demo

👉 https://weather-pulse-flow.lovable.app/

---

## 📊 Data Source

This project uses real-time weather data collected from the **OpenWeatherMap API**.

* Data Type: Live (API-based)
* Location: Tirupati, India
* API: https://openweathermap.org/api

⚠️ Note:
Weather data may slightly vary due to:

* API refresh intervals
* Differences in weather stations
* Network latency

---

## 🔍 Key Features

* 🌡️ Live temperature updates
* 💧 Humidity and wind speed tracking
* 🗺️ Interactive map integration (Leaflet)
* 📊 KPI cards for quick insights
* 📈 Dynamic UI with real-time updates

---

## 📈 Key Insights

* Temperature typically peaks during afternoon hours (12 PM – 3 PM)
* Noticeable drop in temperature during evening hours
* Humidity levels tend to increase during early morning and night
* Real-time data can show slight fluctuations due to continuous updates
